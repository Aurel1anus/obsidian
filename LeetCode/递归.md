#### 什么是递归？
什么是递归呢？函数在运行时调用自己，这个函数就叫递归函数，调用的过程叫做递归。
比如定义函数 f(x)=x+f(x−1)：

```
def f(x) :
		//调用自身
    if x > 0:
		return x + f(x - 1)
		//终止条件
    else : # f(0) = 0
        return 0
```

- 递归函数必须要有**终止条件**，否则会出错；
- 递归函数先不断调用自身，直到遇到终止条件后进行回溯，最终返回答案。
